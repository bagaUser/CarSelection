# Альтернативы py2app для сборки на Python 3.13+

## Рекомендуемые альтернативы

### 1. Briefcase (BeeWare) ⭐ РЕКОМЕНДУЕТСЯ

**Плюсы:**
- ✅ Полная поддержка Python 3.13+
- ✅ Активная разработка и поддержка
- ✅ Отличная работа с Apple Silicon (M1/M2/M3)
- ✅ Создает нативные .app bundles для macOS
- ✅ Хорошая документация

**Минусы:**
- Требует установки дополнительных зависимостей
- Сборка может быть медленнее

#### Установка:
```bash
pip3 install briefcase
```

#### Использование:
См. `setup_briefcase.py` и инструкции ниже.

---

### 2. Nuitka ⭐ ЛУЧШАЯ ПРОИЗВОДИТЕЛЬНОСТЬ

**Плюсы:**
- ✅ Компилирует Python в нативный код
- ✅ Отличная производительность
- ✅ Поддержка Python 3.4-3.13
- ✅ Создает настоящие бинарники
- ✅ Хорошая поддержка Apple Silicon

**Минусы:**
- Может потребовать компилятор C/C++
- Сборка дольше, чем у других инструментов

#### Установка:
```bash
pip3 install nuitka
# Для macOS также нужен компилятор:
xcode-select --install
```

#### Использование:
См. `build_nuitka.sh` ниже.

---

### 3. cx_Freeze

**Плюсы:**
- ✅ Стабильная работа с Python 3.13
- ✅ Простая настройка
- ✅ Кроссплатформенный

**Минусы:**
- Может иметь проблемы с некоторыми GUI библиотеками
- Меньше автоматизации

#### Установка:
```bash
pip3 install cx_Freeze
```

#### Использование:
См. `setup_cxfreeze.py` ниже.

---

## Рекомендация

Для вашего приложения (tkinter + MySQL) рекомендую **Briefcase** - он лучше всего подходит для GUI приложений на macOS и имеет отличную поддержку Python 3.13+ и Apple Silicon.
